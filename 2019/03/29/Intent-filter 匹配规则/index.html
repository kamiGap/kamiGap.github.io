Intent-filter 匹配规则

Mar 29, 2019

Intent-filter 是 activity 在隐式调用的时候使用的一种过滤规则

action、category、data 均命中即可启动 activity

action

​ 是一个 字符串

​ 可以有 多个

​ 只要 一个 匹配即可

category

​ 是一个 字符串

​ 可以有 多个

​ 如果存在 则需要 全部 匹配才算匹配成功

​ 默认会存在一个 android.intent。category.DEFAULT

data

​ 分为两个部分

​ 可以有 多个

​ 只要 一个 匹配即可

URI

​ <scheme>://<host>:<port>/[<path>|<pathPrefix>|<pathPattern>]

​ 满足 scheme 协议的规则

​ URI的默认值为 content 和 file

MimeType

​ 媒体类型

​ 例 图片image/*

​ png image/png

​ 视频video/*

​ mp4 video/mp4

​ 任意媒体格式 */*
